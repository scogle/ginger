--- title: Welcome to Ginger id: welcome ---

Ginger is a utility for quickly generating data-driven static sites. It's intended for journalists, scientists, analysts and others who work with json data to quickly create displays.

The goal is to go from data to display in as little time as possible, while still being flexible enough for refinements and improvements.

What Ginger Isn't

Ginger isn't going to be great for creating complex visualizations or interactive displays. It probably isn't going to be a great solution for big, complex projects that require lots of intircate template features either. Ginger is just meant as a handy tool for quickly turning data into displays.

That being said, feel free to hack away at the layouts and data loader script if you want to do something that isn't supported out of the box. If anything breaks beyond repair, just run ginger new in a new directory and copy your _config.yaml, _views and _data over to start fresh!

Using Ginger

To edit what you see here, modify these sections or create new ones in the _views folder. Views are a combination of instructions about where to get data and a little bit of template code to tell Ginger how to display it.
